# MedWing

**Fully agentic AI platform for medicine delivery in geographically deprived communities.**

MedWing combines voice agents, facial identification, and SLAM mapping to reach rough terrains where traditional medicine delivery fails.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for production

```bash
npm run build
npm run preview
```
